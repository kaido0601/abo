# ABO Log

A static starter blog for the ABO Discord friends group.

## Edit the site

- Change the homepage text in `index.html`.
- Add or edit post pages in the `posts` folder.
- Update colors and layout in `styles.css`.
- Replace `assets/abo-masthead.png` with your own image whenever you want.

## Publish on OVHcloud

This version does not need WordPress, a database, Node.js, or a VPS. Buy a basic OVHcloud web hosting plan that can serve static files, then upload the contents of this folder to the web root, usually `www`.

The file that should load first is `index.html`.
