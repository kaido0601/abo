# ABO Log

A static starter blog for the ABO Discord friends group.

## Edit the site

- Change the homepage text in `index.html`.
- Add or edit post pages in the `posts` folder.
- Update colors and layout in `styles.css`.
- Replace `assets/abo-logo.jpg` whenever the group wants a new logo.

## Publish

This version does not need WordPress, a database, Node.js, or a VPS. It can run on GitHub Pages, Netlify, Cloudflare Pages, or basic static hosting.

The file that should load first is `index.html`.
